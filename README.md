# 🎉 Free Basics Video Download Bot 📲

An innovative app that allows users to download videos using **Facebook Messenger Free Basics** without incurring data charges! 🚀 This bot processes video URLs sent via Messenger and returns the video in text format that fit within Messenger’s size limits.

## ✨ Features

- 🌐 Access through Facebook Messenger Free Basics without data costs.
- 🎥 Download videos by simply sending the video URL.
- 📦 Video is processed and delivered in Messenger-friendly chunks.
- 📲 Perfect for users with limited data access.

🤖 How It Works

    💬 Users send a video URL (e.g., YouTube link) to the bot via Facebook Messenger Free Basics.
    ⚙️ The bot processes the URL and downloads the video.
    ✂️ The video is split into parts that are small enough to be sent through Messenger.
    📤 The bot sends the video parts back to the user.

📚 Technologies Used

    Fastify: Fast and lightweight web framework for Node.js.
    Axios: For interacting with Facebook Messenger's API.
    ytdl-core: For downloading videos from YouTube.

🚧 Future Improvements

    🛠️ Add support for more video platforms beyond YouTube.
    🛑 Better error handling for invalid video URLs.
    🔄 Implement progress tracking for downloads.
